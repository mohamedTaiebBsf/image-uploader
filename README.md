## _Installation_

1. Clone or download this repository.

2. **Before starting the app**, It is **required** to Create `.env` file and define `REACT_APP_BACKEND_URL` environment variable.

   ```sh
   REACT_APP_BACKEND_URL=http://localhost:5000
   ```

3. run:

   ```sh
   $ npm install
   $ npm start
   ```
